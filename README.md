# cpp_note
C++ practise notebook

## Requirements

* Linux : Ubuntu, Debian, CentOS, and others
* macOS(OS X) : may work but not be tested

## Docker

You can also use this in docker, it bases on gcc7/10:

```docker
docker pull chronolaw/cpp_note
docker run -it --rm chronolaw/cpp_note
```
